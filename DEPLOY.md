## 🌐 部署方案（3 选 1）

### 方案 1：Netlify Drop（最简单 ⭐⭐⭐⭐⭐）

**网址：** https://app.netlify.com/drop

**步骤：**
1. 打开上面网址
2. 把 `personal-website` 文件夹拖进去
3. 等待上传完成
4. 获得一个免费域名（如：`yangyuxin-ai.netlify.app`）

**优点：** 最简单，无需注册，1 分钟搞定
**缺点：** 域名较长

---

### 方案 2：Vercel（推荐 ⭐⭐⭐⭐⭐）

**网址：** https://vercel.com

**步骤：**
1. 用 GitHub 账号登录 Vercel
2. 点 "Add New Project"
3. 选择 "Import Git Repository" 或 直接拖拽上传
4. 等待部署完成
5. 获得免费域名（如：`yangyuxin-ai.vercel.app`）

**优点：** 速度快，域名短，支持自定义域名
**缺点：** 需要 GitHub 账号

---

### 方案 3：GitHub Pages（适合长期 ⭐⭐⭐⭐）

**网址：** https://pages.github.com/

**步骤：**
1. 创建 GitHub 仓库（如：`yangyuxin/yangyuxin.github.io`）
2. 上传 `index.html`
3. 开启 GitHub Pages
4. 获得域名（如：`yangyuxin.github.io`）

**优点：** 完全免费，域名专业
**缺点：** 需要 Git 基础

---

## 🎯 推荐方案

**第一次部署：用 Netlify Drop**（最快，1 分钟上线）
**长期使用：用 Vercel**（更专业，支持自定义域名）
